3 Building from the ground up

Letâ€™s start with the simplest linear model: only an intercept. This looks like: \[\begin{equation} y = \beta + \epsilon \tag{3.1} \end{equation}\]

where \(\beta\) is the baseline response, and \(\epsilon\) is residual error. Essentially, weâ€™re predicting the same number for every single measurement. In R syntax, we say that \(y\) is proportional to some value (and estimate the coefficients that make this proportion work out). In this case, \(y\) is proportional to any constant, e.g.Â 1:

\[\begin{equation} y \propto 1 \tag{3.2} \end{equation}\]

Hereâ€™s what it looks like in code:
